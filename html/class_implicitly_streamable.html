Detailed Description

Base class for all classes using the STREAM or STREAM_EXT macros (see Tools/Debugging/ StreamHandler.h) for automatic streaming of class specifications to RobotControl. Only those instances of those classes can be parsed by RobotControl when they are transmitted through MODIFY and SEND macros (see Tools/Debugging/DebugDataTable.h).

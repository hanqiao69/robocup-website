Detailed Description

Formats data as raw text to be used in streams. The physical writing is then done by PhysicalOutStream derivates. Different from OutText, OutTextRaw does not escape spaces and other special characters and no spaces are inserted before numbers. (The result of the OutTextRaw StreamWriter is the same as the result of "std::cout")

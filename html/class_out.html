Detailed Description

The class Out is the abstract base class for all classes that implement writing into streams.

Streaming operator for classes, arrays, and enums. The sizeof expression is evaluated at compile time. It allows distiguishing between a class or array (is not automatically converted to int) and an enum (can be converted to int). The implementation assumes that sizeof(char) != sizeof(short).

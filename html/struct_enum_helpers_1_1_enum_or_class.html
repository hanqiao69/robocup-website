Detailed Description

template<class T, bool isEnum>
class EnumHelpers::EnumOrClass< T, isEnum >

A template class to stream in a different way depending on whether it is instantiated for an enum or a class. This is the version for classes. Here the normal streaming operators are called. To avoid an endless recursion with the template streaming operators below, those operators are defined in the classes In und Out and they are hidden in the derivations In2 and Out2. Since hiding seems to be implemented differently in different compilers, all other streaming operators are defined outside of classes.

Parameters
TThe type of the value to stream.
isEnumIs T an enum type? For this version, this is always false.

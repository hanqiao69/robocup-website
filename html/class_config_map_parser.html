Detailed Description

Parses a ConfigMap with following grammar:

file ::= map

value ::= plain | '[' list ']' | '{' map '}'

plain ::= VALUE

list ::= [ comments value { ',' comments value } [ ',' ] ]

map ::= { comments key '=' value ';' }

comments ::= { COMMENT }

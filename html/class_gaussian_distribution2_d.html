Computes a cholesky decomposition 'A=LLt' for a symmetric positive semidefinite A. If the computation fails, the matrix is not S.P.D. and 'false' is returned. Diagonal entries between [-eps..0] are treated as 0 leading to a valid decomposition. Even diagonal entries infinity are allowed.
